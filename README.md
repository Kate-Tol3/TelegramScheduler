# TelegramScheduler

Telegram-бот для управления уведомлениями и планированием сообщений, который реализует хранение и управление сообщениями в базе данных, запланированную и мгновенную отправку уведомлений, а так же разделение пользователей по ролям и возможность подписки на группы по темам для получения новостей.
Задачи проекта:
•	Проектирование и реализация backend-архитектуры с использованием Kafka, Docker, REST API и базы данных PostgreSQL
•	Разработка логики планирования и обработки сообщений
•	Настройка очередей задач и механизма подписки пользователей по группам
•	Обеспечение корректной работы ролей и доступа к функционалу
Результаты:
•	Создан функциональный Telegram-бот для планирования уведомлений
•	Реализована система асинхронной обработки задач с использованием Kafka
•	Настроено хранение данных и управление пользователями через PostgreSQL
•	Построена backend-часть с использованием REST API и контейнеризации через Docker и оформлена frontend-часть в мессенджере
